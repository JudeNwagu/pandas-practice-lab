# Pandas Practice Lab

A structured hands-on learning repository created to strengthen proficiency in Pandas through focused notebook exercises.

Each notebook isolates one important Pandas concept and demonstrates it through practical examples, making the learning path easy to follow.

This repository is intentionally designed as a **skills lab**, showing confidence in core dataframe manipulation techniques used in real data workflows.

## Learning coverage
This repository covers:

- Pandas Series
- DataFrame attributes and methods
- reading CSV and Excel
- indexing and selection
- sorting
- cleaning and duplicate handling
- whitespace and character cleanup
- string operations
- datetime transformations
- joins and concatenation
- grouping and aggregation
- categorical data
- statistical functions
- plotting

## Why this repository matters
This lab demonstrates practical fluency in Pandas, one of the most important Python libraries for data preparation.

It shows the exact foundational operations used before analysis, dashboards, reporting, and machine learning.

For learners, it serves as a guided progression.

For recruiters, it serves as proof of technical proficiency in dataframe workflows.

## Folder structure
The notebooks are grouped into:
